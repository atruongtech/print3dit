Getting started.

Log your current inventory and you are on your way.

Library Views

Library views offer at-a-glance looks at the whole of your current inventories. Libraries exist for prints, filaments, and printers. Each library view offers filter options which you can use to narrow the number of items on the page. Library views are also mobile-friendly, shrinking the item columns to accomodate a range of screen sizes.

Detail Views

Clicking on an item in a library view will take you into the details view for that item. Here, you can view further information about the print, filament, or printer. You will also find the option to edit the details of items in this view, as well as the option to delete items in the case of prints. You will also find prints that were created using that filament or printer in this view.

Edit and create

The library views naturally start out empty, just waiting to be populated. By clicking the icons within the library views, you can add new items. Populate the fields and upload an image to get your creations and tools into the app. Editing is just as easy, allowing you to correct errors or change up names and other information.
